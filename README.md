# ぐるぐるアバタープログラム

マウスに追従して25方向に振り向く、ブラウザアバター用の最小プログラムです。

## このリポジトリについて

このリポジトリは、元のリポジトリにある仕組み・データを参考にして、自分の画像でぐるぐる動くようにしたものです。

アプリのアイデア、基本的な仕組み、元リポジトリ由来のプログラムやデータを、自分の完全なオリジナルとして主張するものではありません。

自分で作成・追加した部分は基本的に自由に使える形にしていますが、元リポジトリ由来のプログラムコード、データ、構成、素材については、必ず元リポジトリのライセンスと利用条件を優先して遵守してください。

自分が権利を持つ素材、または使用許可を得た素材を `public/slices2/A/` に配置して使ってください。

## セットアップ

必要環境:

- Node.js 22 LTS 推奨
- Vite 8 の要件: Node.js 20.19+ または 22.12+

```bash
npm install
```

## ローカル起動

Windowsなら `start.bat` をダブルクリックすると、ローカルサーバーを起動してブラウザで開きます。

手動で起動する場合は:

```bash
npm run dev
```

手動でアクセスする場合は:

```text
http://127.0.0.1:5173/guruguru.html
```

注意:

- キャラクター素材を入れるまで、プレースホルダーが表示されます。
- マイク、音声ファイル、口パク、まばたき機能は入れていません。

## フレーム画像の配置

このアプリは、キャラクターの向きに応じて `public/slices2/A/` 内の画像を1枚ずつ切り替えます。

画像パス:

```text
public/slices2/A/r{行}c{列}.png
```

例:

```text
public/slices2/A/r2c2.png
```

## 25方向

5列 × 5行の向き差分です。

- 列: 左向き → 正面 → 右向き
  - `c0`: 左向き
  - `c1`: 左斜め
  - `c2`: 正面
  - `c3`: 右斜め
  - `c4`: 右向き
- 行: 上向き → 水平 → 下向き
  - `r0`: 強く上を見る
  - `r1`: 少し上
  - `r2`: 水平
  - `r3`: 少し下
  - `r4`: 強く下

必要なファイルは合計25枚です。

```text
public/slices2/A/r0c0.png ... r4c4.png
```

`src/character-config.js` の `basePath`、`defaultSheet`、`ext` で参照先や画像形式を切り替えできます。

## 自分の素材で作るには

5×5角度シートを1枚用意し、`tools/slice_character_sheets.py` でスライス画像を生成します。

入力ファイル例:

```text
A_目開け_口とじ.png
```

生成後、`public/slices2/A/` に25枚を配置してください。
